Sign Up

All users of the system require a user account with Carma. To create a new user account, the users/registerUser API endpoint must be used. Existing Carma app users may already have an account, and can skip this step, or create an account through the app. A sample of user creation can be seen here. Following client authorization, a client that is looking to sign up new users should do so using the “Register user” endpoint described here. As the endpoint perform registrations for new users and we as a result do not have a user OAuth access token yet, client authentication is performed using plain query parameters in the endpoint.

HTTP Status Code Reason
200 The API call completed successfully
406 email_already_exists|Another user already has the specified email address
