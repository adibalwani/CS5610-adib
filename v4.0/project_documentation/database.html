Database Page

There are three databases at various domains I'm maintaining for my website. They are:
1) Mongo Database hosted on http://carnet-adib.rhcloud.com that holds data for review and favorite.
The schema for favorite collection is: 

{
    userId: String,
    userIdFavorite: String
}

And the schema for review collection is: 
{
    userId: String,
    userIdReview: String,
    rating: String,
    review: String
}

2) The second database is stored at https://api-dev.car.ma Thanks to carma for providing me the required space along with schema development support
3) The third database is stored at https://api.car.ma This is the database I query for getting the search results
