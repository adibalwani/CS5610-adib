Profile Page

Fetch user profile page. Following details are visible on user's profile page :
1) Name
2) Profile picture
3) App Download link
4) Joining date
5) Trip details
6) Review for the user
7) Who favors this user
8) This user's favorite

HTTP Status Code Reason
200 The API call completed successfully
400 invalid_user_id|if the user id provided was invalid
404 not_found|The object could not be found
