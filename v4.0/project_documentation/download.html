Download App

Request a text containing app download information to a telephone number. The input format for body portion of the request is as follows: 

smsAppDownloadDetails {
phoneNumber (string): The user's telephone number, without country code or leading zeroes.,
locale (string, optional): the locale for the message e.g. en or en_US for use english, fr for french fr_CA for canadian french etc.,
clientId (string, optional): the application ouath2 client id, This is not needed when requesting a download as it is part of the request path.,
country (string): The user's country,in ISO2 OR ISO3 format e.g. USA for United States.
}

HTTP Status Code Reason
200 The API call completed successfully.
400 invalid_phone_number|The phone number stored for the user is invalid
