Review Feature

Submit a review for a second person with whom the authenticated user can share a journey. A user can only provide one review for another user. Although the user is allowed to delete already provided review and add a new one. URL for reviews are http://carnet-adib.rhcloud.com Reviews are stored in my database hosted on OpenShift.
