Abstract

Carpooling makes ridesharing easy, safe, and accesible to all. It connects drivers who already have a journey planned, with people who need a ride in the same direction. In just a few seconds, you can offer empty seats in your car, and passengers can book to ride along. By using the Carma app, carpoolers have access to fast and convenient mobility - and can share traveling costs.

At CARNET I want to move the world. Nothing beats the street as infrastructure for individual mobility. But with millions of cars on the road, this asset needs to be better utilized. The carpooling system is designed to do just that, by making ridesharing easy, safe, and convenient for both drivers and passengers.

List of features:

FEATURE USE
Search Used for searching carpool trips as a rider and find information regarding trips for various users
Download Pops up a textbox to enter phone number and a text is sent containing the download link for Carma App
Contact Used for sharing feedback about the site. An email is received by me for each feedback containing the details
Sign Up Used for signing up for my website. My database is created on Carma remote server so uses SHA-256 hash for encoding data
Login Login user into the website. Uses Explicit OAUTH 2.0 for accessing my remote server on Carma
Profile Display profle of a particular user. Profile contains user details, reviews and favorite
Review Provide feedback (review) for another user once the providing user is logged in
Favorite Add/Remove another user from the list of favorites once the user is logged in
Create Trip Create a trip as a driver/rider for recurring days at a particular time leaving from an origin to destination
View Trip View trip details on the search page
