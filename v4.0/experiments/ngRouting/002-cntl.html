This experiment adds profile controller to handle model and view of the profile page. This controller can access the shared LoginService and also accesses the parameters passed when invoking the page using $routeParams. There are two possible ways to obtain username i.e. via passed parameter or via LoginService 

app.controller("ProfileController", function ($scope, LoginService, $routeParams) {
    var username = $routeParams.username;
    $scope.username = username;
});
