About the Experiment

Angular JS Routing is used for changing contents of the page without reloading the page. It follows the MVC pattern in which a controller is defined for specific element and its sub nodes. In this experiment, we have added dependency of ngRoute module. Depending upon differeny URL, contents of the page is updated. 

var app = angular.module("MyApp", ["ngRoute"]);

app.config(['$routeProvider', function ($routeProvider) {
    $routeProvider.
      when('/home', {
          templateUrl: 'partials-000/home.html',
      }).
        when('/profile', {
            templateUrl: 'partials-000/profile.html',
        }).
        when('/about', {
            templateUrl: 'partials-000/about.html',
        }).
      otherwise({
          redirectTo: 'partials-000/home.html'
      });
}]);
