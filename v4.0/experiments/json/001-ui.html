About the Experiment

This experiment demostrates basic jQueryUI operations. Sortable is used for sorting data item in unordered list. Sortable can take object which contains options like grid size, enable, etc. Draggable is used for making the object draggle across the browser. 

$("#sortme").sortable();
$("#dragme").draggable();
