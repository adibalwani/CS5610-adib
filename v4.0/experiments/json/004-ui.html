About the Experiment

This experiment demonstrates use of effect in jQuery UI. effect() function takes four arguments:
1. effect: A string indicating which effect to use for the transition.
2. options: Effect-specific settings and easing.
3. duration: A string or number determining how long the animation will run.
4. complete: A function to call once the animation is complete.
Here we have provided a callback function to be called when effect is completed. 

function callback_fn() {
    setTimeout(function () {
    $("#effect").removeAttr("style").hide().fadeIn();
    }, 500);
};
    
$("#button").click(function () {
    var effect = $("#type").val();
    var options = {};
    $("#effect").effect(effect, options, 500, callback_fn);
});
