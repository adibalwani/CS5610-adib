About the Experiment

This experiment demostrates basic JSON use to read data from webserver, and display the data in a webpage. JSON is a syntax for storing and exchanging data. It is easier and better to use than XML. Here, JSON.parse() is used to get JS object from JSON text. JSON data is stored in a text file and it is an array of objects having property: print and url. Bootstrap is used for table formatting. 

$(function () {
    var xmlhttp = new XMLHttpRequest();
    var url = "000-jsonbasics.txt";
    xmlhttp.onreadystatechange = function () {
        if (xmlhttp.readyState == 4 && xmlhttp.status == 200) {
            var arr = JSON.parse(xmlhttp.responseText);
            display(arr);
        }
    }
    xmlhttp.open("GET", url, true);
    xmlhttp.send();
})
