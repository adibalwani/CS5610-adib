Local Storage

About the Experiment

This experiment demonstrates use of local storage on browser to save and load data. localStorage's setItem() function is used to store a value and it takes input a key value pair. getItem() is used to get a value by providing the key. 

function save() {
    var val = $("#input").val();
    localStorage.setItem("data", val);
}
function load() {
    $("#existing").text(localStorage.getItem("data"));
}
