About the Experiment

This experiment demonstrates JSONP to access JSON object from web server that is other than the origin of the page. CORS allows to access JSON data on other site without the need for being the same page where response wass generated. Also, I have experimented on dragging one DOM object over other. 

function allowDrop(ev) {
    ev.preventDefault();
}
function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id);
}
function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
}
