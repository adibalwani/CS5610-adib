About the Experiment

This experiments demonstrates use of Node JS to serve request of clients. There can different types of request based on CRUD operations. For Create we use post, for read we use get, for update we use put and for delete we use delete. These standards are followed in RestAPI. Also, depending upon the hierarachy, we can request JSON object starting with root followed by index number if we want specific object. 

    $http.get("/login")
    .success(function (response) {
        $scope.login = response;
    });

    $scope.update = function (login) {
        $http.put("/login/" + $scope.index, login)
        .success(function (response) {
            $scope.login = response;
        });
    };
