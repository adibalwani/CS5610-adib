Date Picker

About the Experiment

This experiment demonstrates use of ui.bootstrap module in angular to display date. User can navigate through months and years. The datepicker shows dates that come from other than the main month being displayed. These other dates are also selectable. Also, a min date as current date is set. It is used in my project for selecting the date of journey for the carpool. 

    $scope.open = function ($event) {
        $event.preventDefault();
        $event.stopPropagation();
        $scope.minDate = $scope.minDate ? null : new Date();
        $scope.opened = true;
    };
