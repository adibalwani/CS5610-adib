2. HTML Text Formatting

<p> This is a paragraph. Any empty line is added before and after a paragraph. Any number of spaces and new lines, count as one space.

<br> If you want a line break without starting a new paragraph. Example:
I am in the same paragraph

I am in new paragraph

<pre> Defines a block of pre-formatted text. It keeps spacings and line-breaks intact. Example:

This space                 is maintained
                along with line-break

<b> This is a bold text

<strong> This is a strong text with semantic importance

<i> This is an italic text

<em> This is an emphasised text with semantic importance

<mark> This is a marked text

<abbr> Abbrevate: NEU

<address> Address:
