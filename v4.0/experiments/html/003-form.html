About the Experiment

This experiment is used to obtain user data. It also demonstrates the data-time picker of HTML5. Label element is used to focus on associated element of it and also give a label to that element. Date is chosen using datetime-local type introduced in HTML5. Datalist of universities is created. 

<div>
    <label for="dos">Date of Start: </label>
    <input id="dos" type="datetime-local" name="dos" />
</div>
