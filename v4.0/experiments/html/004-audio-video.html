About the Experiment

This experiment is used to embed youtube video onto the page. Also, audio can be included. HTML5 supports direct support for playing youtube videos using iframes. Audio is played using <audio> tag. src attribute is used to get the source of the audio/video. <audio controls> is used to give a controller to the audio. Youtube video is embed in an iframe. 

<audio controls>
    <source src="http://ia600402.us.archive.org/29/items/MLKDream/MLKDream_64kb.mp3" />
    Your browser doesnot support the audio element
</audio>

<iframe width="300" height="300" src="http://www.youtube.com/embed/NmugSMBh_iI"></iframe>
