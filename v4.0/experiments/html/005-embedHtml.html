Embed HTML Page

About the Experiment

This experiment demonstrates how to embed an html page into other. <embed> tag is used along with the source URL for embedding other page into this one. EmbeddingHTML is useful if we want to display other HTML pages like ads without getting the source code. 

<embed width=250px height=100px src="000-helloworld.html" />
