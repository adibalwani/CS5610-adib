2. HTML Block Element

Block level elements normally start (and end) with a new line, when displayed in a browser. Inline elements are normally displayed without line breaks

<div> It is a block level element and can be used as a container for other elements

<span> It is an inline element and can be used for text highlighting

3. HTML Iframe

Iframe is used to display other HTML page. It is like embedding that page inside the current one.
