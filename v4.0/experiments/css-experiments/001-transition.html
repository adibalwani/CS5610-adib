Hover over the Home to display its content

HOME

Welcome to the home tab. The transition effect is performed for each menu option.

About the Experiment

This experiment is used to display menu when mouse hovers over the menu tag. The transition effect is obtained by using hover in CSS. The details to be displayed are kept hidden with opacity of 0 and when he mouse hovers over the menu tab, the opacity is made to 1. The transition setting is kept to two seconds. 

#home {
    opacity: 0;
    transition: opacity 1s linear;
}

#option:hover + #home {
    opacity: 1;
}
