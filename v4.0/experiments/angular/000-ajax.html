AJAX with JSONP

Please wait some time if you have slow internet

    About the Experiment

    An asynchronous request doesn’t block the client i.e. browser is responsive. At that time, user can perform another operations also. In such case, javascript engine of the browser is not blocked. This experiment demonstrates use of AJAX to get JSONP data from IMDB and display it on user screen. 

    $.ajax({
    url: "http://www.myapifilms.com/imdb?title=" + $("#movie").val() +
        "&format=JSONP&aka=0&business=0&seasons=0&seasonYear=0&technical=0&
        filter=N&exactFilter=0&limit=10&lang=en-us&actors=N&biography=0&trailer=0&
        uniqueName=0&filmography=0&bornDied=0&starSign=0&actorActress=0&actorTrivia=0&
        movieTrivia=0&awards=0&moviePhotos=N&movieVideos=N",
    dataType: "jsonp",
    success: renderMovies
});
