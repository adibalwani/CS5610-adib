Form Validation

About the Experiment

This experiment demonstrates use of JavaScript/jQuery for validation of form on client side itself, thereby increasing efficiency. Whenever a key event occurs for any input field its corresponding validation method is called. Date picker responds to mouse picker too For username, the length of input is checked to be greater than five. For password, along with the condition of having length greater than five, a digit is required too. Regural expression comes handy in forming such conditions. Current date is compared with the input date for DOB and if such date doesnot exist error is thrown. Bootstrap is used for styling the form. 

function validateUsername() {
    var username = $("#username").val();
    if (username == null || username == "" || username.length < 6) {
        $("#username_msg").html("<img src='incorrect.png' width='20' height='20'/>" +
          " Username must be at least 6 char long");
        return false;
    } else {
        $("#username_msg").html("<img src='correct.png' width='20' height='20'/>");
        return true;
    }
}
