About the Experiment

This experiments demonstrates the use of carousel in Bootstrap framework. Carousel is used for displaying various images to choose for the background. Once the user decides which image to choose he clicks button to set that as the background image for that user. User can rechange the background as many times the user wants. For retriving the current image as active I use div.active and then I get the image source via few more functions. 

function changeBackground() {
    var image = $('div.active').html();
    image = $(image).attr("src");
    $("body").attr("style", "background-image:url(" + image + ")"); //change background
}
