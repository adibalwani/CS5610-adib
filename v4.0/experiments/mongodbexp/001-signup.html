About the Experiment

Username: adib Password: adib123 This experiment demonstrates the use of ui-bootstrap an Angular JS module for modal which is like a pop up used for signup form. Whenever a button click event is generated modal is open and used to display a form of signup. Also, various checking are added in the experiment for form validation. This doesn't send a request to the server to save. Next, experiment shows that. 

    $scope.signUp = function () {
        var modalInstance = $modal.open({
            templateUrl: 'partials-001/signup.html',
            controller: 'SignupController'
        });
    }
